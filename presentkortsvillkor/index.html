Windskore – Presentkortsvillkor

/presentkortsvillkor
Senast uppdaterad: 2026-07-21

När du köper ett presentkort från Windskore godkänner du dessa villkor. Läs igenom villkoren innan du genomför ditt köp.

1. Om Windskore Presentkort

Windskore Presentkort är ett digitalt eller fysiskt värdebevis som kan användas för köp av produkter och tjänster som erbjuds av Windskore. Presentkortet har ett bestämt värde och kan endast användas hos Windskore enligt de regler som anges i dessa villkor.

2. Köp av presentkort

När du köper ett Windskore Presentkort ansvarar du för att de uppgifter du lämnar vid köpet är korrekta. Efter genomfört köp skickas presentkortet enligt den leveransmetod som valts:

  • Digitala presentkort skickas via e-post eller visas på kundens konto.
  • Fysiska presentkort skickas enligt vald leveransmetod.

Windskore ansvarar inte för förseningar som orsakas av felaktiga kunduppgifter.

3. Användning av presentkort

Presentkortet kan endast användas på de platser där Windskore anger att presentkort accepteras. Presentkortet:

  • Kan inte användas flera gånger även om hela värdet inte är förbrukat.
  • Kan inte användas för köp utanför Windskore.
  • Kan inte lösas in mot kontanter.

4. Giltighetstid

Windskore Presentkort gäller fram till det datum som anges på presentkortet. Efter att giltighetstiden på presentkortet har gått ut kan presentkortet inte längre användas och eventuellt kvarvarande saldo kan inte nyttjas.

Kunden ansvarar själv för att kontrollera giltighetsdatumet på sitt presentkort. Om ingen giltighetstid anges på presentkortet gäller reglerna som följer av svensk konsumenträtt.

5. Förlorade eller stulna presentkort

Ett presentkort ska behandlas som en värdehandling. Windskore ansvarar inte för förlorade, stulna eller obehörigt använda presentkort om kunden inte kan visa att ett tekniskt fel hos Windskore har orsakat problemet.

6. Återbetalning

Köp av presentkort kan omfattas av ångerrätt vid distansköp enligt gällande lagstiftning. Om presentkortet helt eller delvis har använts kan särskilda regler gälla. Ett presentkort som redan har använts kan inte återbetalas.

7. Bedrägeri och missbruk

Windskore har rätt att spärra presentkort som misstänks ha:

  • blivit stulna,
  • skapats genom obehörig åtkomst,
  • använts på ett sätt som bryter mot dessa villkor.

8. Ändringar av villkor

Windskore kan uppdatera dessa villkor vid behov. Ändringar påverkar inte redan köpta presentkort om inget annat krävs enligt lag.
