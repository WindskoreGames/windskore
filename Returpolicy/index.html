Returpolicy för Windskore

/returpolicy
Gäller från: 21 maj 2026

Denna returpolicy gäller för köp gjorda via Windskore, inklusive spel, digitala produkter, tjänster och eventuella fysiska produkter kopplade till Windskore.

Genom att genomföra ett köp godkänner du denna returpolicy.

1. Digitala produkter

Alla köp av digitala produkter är slutgiltiga efter genomförd betalning.

Återbetalning ges endast vid tekniska fel som gör produkten obrukbar och inte kan lösas inom rimlig tid.

Vid felaktigt köp måste kontakt tas inom 24 timmar för bedömning.

2. Fysiska produkter

Du har rätt att returnera varor inom 14 dagar från mottagandet.

Produkten måste vara i originalskick och får inte vara använd på ett sätt som påverkar värdet.

Kunden står för returfrakten om inte produkten är felaktig.

3. Återbetalningar

Godkända återbetalningar behandlas inom 5–10 arbetsdagar.

Återbetalning sker via samma betalmetod som användes vid köpet.

Eventuella avgifter från betalningsleverantörer återbetalas inte.

4. Felaktiga eller trasiga produkter

Kontakta support inom 7 dagar efter leverans.

Bifoga bilder eller tydlig felbeskrivning.

Vi erbjuder ersättning, reparation eller återbetalning beroende på situation.

5. Missbruk och bedrägeri

Windskore förbehåller sig rätten att neka återbetalning vid misstänkt missbruk.

Detta gäller även om produkten redan använts eller om användarvillkor brutits.

6. Kontakt
